РАБОТА С ТЕРМИНАЛОМ. ПЕРВЫЕ КОМАНДЫ:

	man			- помощь
	info 			- тоже помощь
	uptime 			- время с последнего включения
	lscpu 			- основные данные процессора
	whatis 			- показывет, что делает команда
	whereis 		- показывает, где файл
	locate 			- показывает, где файл
	ls 			- показывает, что находится в директории

		-R - recursive

	Ctrl + Z 		- отправить процесс в фон
	Ctrl + C 		- прекратить процесс
	
НАВИГАЦИЯ ПО ФАЙЛАМ И ДИРЕКТОРИЯМ:

	cd 			- сменить директорию
	ls 			- вывести содержимое директории
	pwd 			- вывести данный путь (показать где я)
	~ 			- сокращениие директории home
	/ 			- коренная директория Linux (корень системы)
	.. 			- директория выше
	. 			- данная директория (где я сейчас)
	
СОЗДАНИЕ, КОПИРОВАНИЕ, ПЕРЕНОС, ПЕРЕИМЕНОВАНИЕ, СТИРАНИЕ ФАЙЛОВ:

	touch 			- создать файл или обновить время создания
	cp 			- скопировать файл
	mv 			- перенести или переименовать файл
	rm 			- стереть файл
	
РАБОТА С ДИРЕКТОРИЯМИ:

	mkdir 			- создать директорию
	rmdir 			- стереть пустую директорию
	cp 			- копировать директорию
	mv 			- переименовать или перенести директорию
	rm -R 			- стереть не пустую директорию со всем содержимым

	sudo rm -R / 		- удалить корень системы Linux (вызовет сбой, удаление системы)
	
СОЗДАНИЕ ЛИНКОВ (ССЫЛОК НА ОБЪЕКТ, ФАЙЛ ИЛИ ДИРЕКТОРИЮ):
	
	ln 			- создать дубликат файла
	ln -s 			- создать symbolic линк на файл или директорию типа shortcut
	
КОМАНДЫ FIND, CUT, SORT, WC:

	find 			- найти файл
	wc 			- вывести количество строк, слов, байт
	cut 			- вывести определенное поле из текста
	sort 			- вывести отсортированный текст (по какому-то признаку)
	
КОМАНДА GREP:

	grep 			- поиск определенного слова в файле и вывод строк с этим словом
	
РЕГУЛЯРНЫЕ ВЫРАЖЕНИЯ:

	[A-Z]* 			- любое слово из больших букв
	[0-9]* 			- любое количество подряд стоящих цифр
	[A-Za-z]*@[A-Za-z]*.com - простое выражение емэйлов с окончанием .com
	www\.[a-z]*\.com 	- любой веб-адрес с окончанием .com
	
ПЕРЕНАПРАВЛЕНИЕ ВЫВОДА/ВВОДА И /DEV/NULL:

	> 			- перенаправление "хороших" результатов в новый файл (перезапись старого)
	>> 			- перенаправление "хороших" результатов с добавлением в новый или старый файл
	
	2> 			- перенаправление "плохих" результатов в новый файл (перезапись старого)
	2>> 			- перенаправление "плохих" результатов с добавлением в новый или старый файл
	
	&> 			- и "плохое" и "хорошее" в файл
	
	/dev/null 		- устройство находящиеся "нигде" (мниое, несуществующее устройство (место))
	
АРХИВИРОВАНИЕ И СЖАТИЕ:

	tar cf mytar.tar Folder1 	- заархивировать Folder1 в архив mytar.tar
	tar xf mytar.tar 		- разархивировать архив mytar.tar
	
	gzip 				- сжать файл архиватором gzip
	bzip2 				- сжать файл архиватором bzip (сжимает/компрессирует лучше всего)
	xz 				- сжать файл архиватором xzip
	
	gunzip | bunzip2 | unxz 	- распаковать файл соответствующим архиватором
	
	tar cvzf myBZIP2.bz2 Folder1 	- сжать Folder1
		флаг v			- visualisate - показ всех действий по архивированию
		флаг f			- file - всегда идет последним
	tar xvf myBZIP2.bz2 		- распаковать архив
	tar tf myBZIP2.bz2 		- посмотреть (вывести) содержимое архива
	
	zip -r myZIP.zip Folder1 	- запаковать Folder1 в ZIP
		флаг -r			- recursive - со всем содержимым внутри
	unzip myZIP.zip 		- распаковать файл myZIP.zip

	top 				- как task manager в windows, показать все текущие процессы, загруженность процессора и памяти
	ps 				- показать мои процессы

	free 				- показать состояние памяти в байтах
	free - h 			- показать состояние памяти в МБайтах, ГБайтах

АККАУНТЫ И ВАЖНЫЕ ФАЙЛЫ АККАУНТОВ:
	
	По умолчанию в Linux есть 3 группы аккаунтов: 
		root (может делать всё без ограничений)
		administrator (пользователь, имеющий root-права)
		standart (пользователь, не имеющий root-прав)

	sudo				- запустить комманду, используя права Super User
	su				- сменить текущего пользователя (su + имя_пользователя)
	
	/etc/passwd			- файл со всеми аккаунтами
	/etc/shadow			- файл со всеми паролями (зашифрованными) аккаунтов
	/etc/group			- файл со всеми группами аккаунтов
	
	whoami 				- показать имя текущего пользователя
	id				- показать группы принадлежности данного пользователя
	who				- показать, кто сейчас в системе
	w				- показать, кто сейчас в системе и что делает (какие действия запущены)
	last				- показать последние входы в систему
	
ПОЛЬЗОВАТЕЛИ И ГРУППЫ:

	/etc/skel/			- шаблон для нового пользователя (все, что есть в данной директории /etc/skel/ будет автоматически добавляться в домашнюю директорию нового пользователя при его создании. /etc/skel/ представляет некий шаблон для каждого нового пользователя и автоматически будет копироваться в его домшнюю директорию)
	
	useradd -m user1		- создать нового пользователя с именем user1 
		флаг -m			- автоматически создать пользователю домашнюю директорию и копировать в нее содержимое из /etc/skel/
	userdel user1			- удалит пользователя user1, но оставить его домашнюю директорию
	userdel -r user1		- удалит пользователя user1 и всё содержимое его домашней папки (также удаляет его из /etc/passwd)
		флаг -r			- recursive - удалить со всем содержимым
	
	groupadd имя_группы		- создать группу пользователей имя_группы
	groupdel			- удалить группу пользователей
	
	usermod -aG имя_группы имя_пользователя		- добавить в группу пользователя
		флаг -a					- append - добавить
		флаг -G					- group - добавить в группу
		
	usermod -aG sudo имя_пользователя		- добавить в группу sudo пользователя, автоматически делает пользователя администратором 
	
	deluser имя_пользователя имя_группы		- удалить пользователя из группы (если удалить пользователя из группы sudo, то с него снимаются права администратора)
